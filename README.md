# Security Writeups

This repository will contain security writeups and notes from platforms like TryHackMe, homelab investigations, and other hands-on exercises.

Each writeup will focus on:
- The scenario or room (what the environment is simulating)
- The steps I took to investigate or solve it
- Key findings, indicators, and what I would report in a SOC context
- Lessons learned and how they apply to real-world security operations

The goal is to show my **analysis process** and how I think through alerts, logs, and incidents — not just the final answer.
